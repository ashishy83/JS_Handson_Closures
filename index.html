Q2. Print the Output.

        let count = 0;
        (function () {
        if (count === 0) {
            let count = 1;
            console.log(count); // What is logged?
        }
        console.log(count); // What is logged?
        })();

Output:
1
0

Explanation: let is block scoped variable, so for first console the scope of count is inside if hence 1 will be logged and for second console the scope of count is the global declared count hence 0 is logged

Q7. Print the output. 

 
        var a = 10;
        var x = (function () {
            var a = 12;
            return function () {
            alert(a);
            };
        })();
        x();

Output: 12

Explanation:Here "a" is forming a closure with the inner function,so it can be accessed inside inner function, hence getting 12 as the output.
